### Definition
Eine Funktion ist eine mathematische Abbildung, die jedem Element einer Definitionsmenge genau ein Element einer Zielmenge zuordnet.
#### Definitionsbereich
Der Definitionsbereich einer Funktion ist die Menge aller Eingabewerte, für die die Funktion definiert ist.
In anderen Worten, es ist die Menge aller möglichen Werte, die man in die Funktion einsetzen kann,
ohne auf Probleme wie die Division durch Null oder das Ziehen der Wurzel aus einer negativen Zahl zu stoßen.<br />
$D(f) = [5, 40]$
#### Wertebereich
Der Wertebereich einer Funktion ist die Menge aller Ausgabewerte, die die Funktion annehmen kann,
wenn man alle möglichen Werte aus ihrem Definitionsbereich einsetzt.<br />
$W(f) = [0, 250]$

### Lineare Funktionen
#### Grundfunktion
$f(x) = m * x + q$
- **m** = Steigung, **q** = y-Achsenabschnitt

#### Steigung *m* berechnen
$m = \dfrac{\Delta y}{\Delta x} = \dfrac{y2 - y1}{x2 - x1}$

- Funktion heisst **steigen** wenn f(x) grösser wird wenn x grösser wird.
- Funktion heisst **fallen** wenn f(x) kleiner wird wenn x grösser wird.
- Funktion heisst **monoton** wenn sie weder steigt noch fällt.

#### Der Differentialquotient
$\dfrac{\Delta y}{\Delta x} = \dfrac{f(x + \Delta x) - f(x)}{\Delta x}$

#### Nullstelle
Die Nullstelle ist der Schnittpunkt der Gerade auf der x-Achse -> $y = 0$

#### Gerade rechtwinklig zu einer anderen
Für die Berechnung einer Gerade, welche rechtwinklig zu einer andere ist, gilt:<br />
$m2 = -\dfrac{1}{m1}$

#### Mittelpunkt zwischen zwei Geraden
Um den Mittelpunkt zwischen zwei Punkten zu berechnen gilt:<b />
$M = \dfrac{a1 + x2}{2} : \dfrac{y1 + y2}{2}$

#### Proportionalität
Eine Funktion ist proportional, wenn sie durch den Ursprung geht. Das bedeutet, dass der y-Achsenabschnitt 0 ist.

#### Gleichung allgemein Quadratisch
$a * x^2 + b * x + c = 0$
#### Mitternachtsformel
$x1, x2 = \dfrac{-b(+/-)\sqrt{b^2 -4ac}}{2a}$
#### Lösen mit Faktorzerlegung
Die Gleichung muss in die Form: $(x + 4)(x - 8) = 0$ gebracht werden. Lösungen sind -4 und 8.
#### Diskriminante
$D = Diskriminante = b^2 - 4ac$ <br />
$D > 0$ -> zwei Lösungen <br />
$D = 0$ -> eine Lösung <br />
$D < 0$ -> keine lösung <br />
