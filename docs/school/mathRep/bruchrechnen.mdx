### Kürzen von Brüchen
<HighlightBox> **Kürzen=** Streichen von gemeinsamen Faktoren in Zähler und Nenner </HighlightBox>
- $\dfrac{24}{36} = \dfrac{2*2*2*3}{2*2*3*3} = \dfrac{2}{3}$

<HighlightBox> **Nie aus Summen kürzen!** Summen immer zuerst in Produkte umwandeln </HighlightBox>
- $\dfrac{a}{ab+ac} = \dfrac{a}{a*(b+c)} + \dfrac{1}{b+c}$

### Das Minus bei Brüchen
<HighlightBox> Minus kann verschoben werden </HighlightBox>
- $-\dfrac{a}{b} = \dfrac{-a}{b} = \dfrac{a}{-b}$
- $-\dfrac{a+b}{c+d} = \dfrac{-(a+b)}{c+d} = \dfrac{a+b}{-(c+d)}$

### Erweitern von Brüchen
<HighlightBox> **Erweitern=** Zähler und Nenner mit der gleichen Zahl multiplizieren </HighlightBox>
- $\dfrac{3}{4} =  \dfrac{3(*2)}{4(*2)} = \dfrac{6}{8}$

### Multiplikation von Brüchen
<HighlightBox> Zähler * Zähler und Nenner * Nenner </HighlightBox>
- $\dfrac{3}{4} * \dfrac{5}{6} = \dfrac{3*5}{4*6} = \dfrac{15}{24}$

### Addition und Subtraktion von Brüchen
<HighlightBox> Bei gleichen nennern kann Zähler plus/minus Zähler gerechnet werden </HighlightBox>
- $\dfrac{a}{b} + \dfrac{c}{b} = \dfrac{a+c}{b}$

<HighlightBox> Klammerregeln beachten </HighlightBox>
- $\dfrac{a}{d} - \dfrac{b+c}{d} = \dfrac{a-b-c}{d}$

<HighlightBox> Bei ungleichen Nennern müssen die Brüche zuerst auf den gleichen Nenner gebracht werden </HighlightBox>
- $\dfrac{a}{b} + \dfrac{c}{d} = \dfrac{ad+bc}{bd}$
### Division von Brüchen
<HighlightBox> Dividiert wird, indem man mit dem Kehrbruch multipliziert  </HighlightBox>
- $\dfrac{a}{b} : \dfrac{c}{d} = \dfrac{a}{b} * \dfrac{d}{c} = \dfrac{ad}{bc}$

### Doppelbrüche umformen
<HighlightBox> Den Hauptbruch durch ein Divisionszeichen ersetzen  </HighlightBox>
- $\dfrac{\dfrac{a}{b}}{\dfrac{c}{d}} = \dfrac{a}{b} : \dfrac{c}{d}$

<HighlightBox> Wenn der Zähler oder Nenner zwar aussieht wie ein Bruch, aber tatsächlich eine Summe (Differenz) ist, muss man  diese  zuerst  zu  einem Bruch  machen .  </HighlightBox>
$\dfrac{a+\dfrac{1}{2}}{b - \dfrac{3}{4}} = \dfrac{\dfrac{2a+1}{2}}{\dfrac{4b-3}{4}} = \dfrac{2a+1}{2} : \dfrac{4b-3}{4}$
