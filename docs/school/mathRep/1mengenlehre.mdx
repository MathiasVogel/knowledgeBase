### Elemente einer Menge
- $x \in A \Rightarrow$ x ist Element der Menge A
- $x \notin A \Rightarrow$ x ist kein Element der Menge A

### Begriffe
- Anzahl Elemente

  - Zwei Mengen A und B heissen **gleichm√§chtig**, wenn sie die **gleiche** Anzahl Elemente haben.

  - Zwei Mengen A und B heissen **gleich**, wenn sie die genau gleichen Elemente besitzen. $A = B$
