title: Bruchrechnen
---

### Kürzen von Brüchen
:::info
**Kürzen=** Streichen von gemeinsamen Faktoren in Zähler und Nenner.
:::
- $\dfrac{24}{36} = \dfrac{2*2*2*3}{2*2*3*3} = \dfrac{2}{3}$

:::info
**Nie aus Summen kürzen!** Summen immer zuerst in Produkte umwandeln.
:::
- $\dfrac{a}{ab+ac} = \dfrac{a}{a*(b+c)} + \dfrac{1}{b+c}$

### Das Minus bei Brüchen
:::info
Minus kann verschoben werden.
:::
- $-\dfrac{a}{b} = \dfrac{-a}{b} = \dfrac{a}{-b}$
- $-\dfrac{a+b}{c+d} = \dfrac{-(a+b)}{c+d} = \dfrac{a+b}{-(c+d)}$

### Erweitern von Brüchen
:::info
**Erweitern=** Zähler und Nenner mit der gleichen Zahl multiplizieren.
:::
- $\dfrac{3}{4} =  \dfrac{3(*2)}{4(*2)} = \dfrac{6}{8}$

### Multiplikation von Brüchen
:::info
Zähler * Zähler und Nenner * Nenner.
:::
- $\dfrac{3}{4} * \dfrac{5}{6} = \dfrac{3*5}{4*6} = \dfrac{15}{24}$

### Addition und Subtraktion von Brüchen
:::info
Bei gleichen nennern kann Zähler plus/minus Zähler gerechnet werden.
:::
- $\dfrac{a}{b} + \dfrac{c}{b} = \dfrac{a+c}{b}$

:::info
Klammerregeln beachten.
:::
- $\dfrac{a}{d} - \dfrac{b+c}{d} = \dfrac{a-b-c}{d}$

:::info
Bei ungleichen Nennern müssen die Brüche zuerst auf den gleichen Nenner gebracht werden.
:::
- $\dfrac{a}{b} + \dfrac{c}{d} = \dfrac{ad+bc}{bd}$

### Division von Brüchen
:::info
dividiert wird, indem man mit dem Kehrbruch multipliziert.
:::
- $\dfrac{a}{b} : \dfrac{c}{d} = \dfrac{a}{b} * \dfrac{d}{c} = \dfrac{ad}{bc}$

### Doppelbrüche umformen
:::info
Den Hauptbruch durch ein Divisionszeichen ersetzen.
:::
- $\dfrac{\dfrac{a}{b}}{\dfrac{c}{d}} = \dfrac{a}{b} : \dfrac{c}{d}$

:::info
Wenn der Zähler oder Nenner zwar aussieht wie ein Bruch, aber tatsächlich eine Summe (Differenz) ist, muss man diese zuerst zu einem Bruch machen.
:::
$\dfrac{a+\dfrac{1}{2}}{b - \dfrac{3}{4}} = \dfrac{\dfrac{2a+1}{2}}{\dfrac{4b-3}{4}} = \dfrac{2a+1}{2} : \dfrac{4b-3}{4}$
