### Potenzrechnen
:::info
$\Rightarrow$ Im Ausdruck b = a heisst $a^n$ die Basis, n der Exponent und a Potenzwert oder n-te Potenz von a.
:::

#### Potenzgesetze
- $a^n * a^n = (ab)^n$
- $\dfrac{a^n}{b^n} = (\dfrac{a}{b})^n$
- $a^0 = 1$
- $a^n * a^m = a^{n+m}$
- $a^{-n} = \dfrac{1}{a^n}$
- $\frac{a^n}{a^m} = a^{n-m}$
- $(a^n)^m = a^{n*m}$
- $a^{\frac{1}{n}} = \sqrt[n]{a}$
- $a^{\frac{n}{m}} = \sqrt[m]{a^n}$
- $a^{\frac{n}{m}} = (\sqrt[m]{a})^n$

### Radizieren
:::info
$\Rightarrow$ Im Ausdruck $\sqrt[n]{a}$ heisst a Radiant und n die Wurzelexponent.
:::
#### Radiziergesetze
- $\sqrt{a}^2 = \sqrt{a} * \sqrt{a} = a$
- $\sqrt{a} * \sqrt{b} = \sqrt{a*b}$
- $\sqrt{\dfrac{a}{b}} = \dfrac{\sqrt{a}}{\sqrt{b}}$
- $\sqrt[n]{\dfrac{a}{b}} = \dfrac{\sqrt[n]{a}}{\sqrt[n]{b}}$
- $(\sqrt[n]{a})^m = \sqrt[n]{a^m}$
- $a\sqrt[n]{a} + b\sqrt[n]{b} = (a+b)\sqrt[n]{a}$
- $\sqrt[m]{\sqrt[n]{a}} = \sqrt[m*n]{a}$
- $\sqrt{a} + \sqrt{b} \neq \sqrt{a+b}$
- $\sqrt{a} = a^{\frac{1}{2}}$
- $\sqrt[n]{a} = a^{\frac{1}{n}}$
- $\sqrt[n]{a^m} = a^{\frac{m}{n}}$
